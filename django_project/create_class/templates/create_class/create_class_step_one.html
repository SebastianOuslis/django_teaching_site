Create Class Step 1

There are three types of classes you can have on NxtKlass: One on One Video Calls, Live-streamed Classes, Pre-uploaded videos
One on One:
These classes are meant to replicate a one on one setting with a student. This allows for the most connection between the instructor and the student. When the class starts, the teacher and the student will have a video call together.
This type of class can have a maximum of one student at a time. The first person to purchase the class will be the student.
Create a One on One Class

Live-stream: ** Currently In Development **
As of currently, this feature is not possible yet. We are working hard to add this to the site
Pre-uploaded videos:
These classes are meant for instructors who have already recorded their classes and want students to watch these videos.
Students that have purchased the class can watch the video at any time.
